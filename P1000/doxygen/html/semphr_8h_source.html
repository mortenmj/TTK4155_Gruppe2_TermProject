289 /*
290  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
291  *
292  * The source code that implements the alternative (Alt) API is much
293  * simpler because it executes everything from within a critical section.
294  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
295  * preferred fully featured API too. The fully featured API has more
296  * complex code that takes longer to execute, but makes much less use of
297  * critical sections. Therefore the alternative API sacrifices interrupt
298  * responsiveness to gain execution speed, whereas the fully featured API
299  * sacrifices execution speed to ensure better interrupt responsiveness.
300  */
301 #define xSemaphoreAltTake( xSemaphore, xBlockTime ) xQueueAltGenericReceive( ( xQueueHandle ) ( xSemaphore ), NULL, ( xBlockTime ), pdFALSE )
