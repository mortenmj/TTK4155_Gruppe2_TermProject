void adc_start_conversion ( void  )

Start a new round of conversions.

The ADC interrupt will iterate through all available ADC channels and queue the results. Once the process is complete, the interrupt will give a semaphore which can be claimed with adc_take_semaphore () so that the queued values may be retrieved with adc_get_value ().
